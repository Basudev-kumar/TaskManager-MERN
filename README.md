# Task Manager (MERN Stack)

A **Task Management System** built using the **MERN Stack (MongoDB, Express.js, React, Node.js)** that provides role-based access (Admin/User), dynamic dashboards, multi-user task assignments, file uploads, and Excel report generation.


(Screenshot 2025-07-06 030915.png)

---

## 🌟 Features
- 🔐 **User Authentication & Authorization** (JWT-based)
- 🛡️ Role-based access (Admin/User)
- ✅ Task creation, editing, deletion, and status updates
- 📋 Assign tasks to multiple users
- 📊 Admin and User-specific dashboards with statistics
- 🗂️ To-do checklist inside tasks with progress tracking
- 🗓️ Task deadlines and priority management
- 📎 File upload for profile pictures and task attachments
- 📥 Export tasks and user-task reports to Excel files
- 🛠️ RESTful API structure
- 📈 Real-time UI updates with toast notifications (frontend)

---

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **File Uploads:** Multer
- **Data Export:**
- **Version Control:** Git & GitHub
- **Deployment:** Vercel (Frontend) & Render (Backend)
